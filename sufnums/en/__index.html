Welcome to the random number suffixal generator!

How to use?

What is random number suffixal generator?

Random number suffixal generator is a program which generates a random number in a user-defined range and adds to this number user specified suffix and/or ending.

123
.abc
.jpg

Why was random number suffixal generator created?

Random number suffixal generator was created in order to help web developers automate the process of naming the new image file.

Where did the idea come from?

One of the most common operations performed in web development is the creation and optimization of images for the site. Especially if we are talking about support for existing resources, where you want to add or change images. In this case, the developer will encounter with the procedure of naming an image file, i.e. will have to give a name to this file. When such files are few it’s not difficult, for example logo.png or Winnie_the_Pooh.jpg. But if we are talking about adding a new image into the existing large database of images that are used on various pages of the website, then definitely the question arises: What name to give the image file? After all, we have our database of hundreds different files and a file with the same name already can easily exist there.

One way to solve this problem is to give your new file name a random number. Exactly for this random number suffixal generator was created.

Attention! This method of naming image is suitable only when you don't need too care about image optimization for better indexing by search engines.

Why do we need suffixes?

The suffix added to number is an additional classification for easy sorting of image files. For example, the suffix ".a" indicates that this image belongs to the class of "article" and will be placed on the website in some articles, and the suffix ".s" indicates that this image belongs to the class of "sidebar" and will be posted on the sidebar of the website. You can come up with your own suffix, putting it in the "Add random suffix" in the configuration panel.

Do not judge strictly, but this generator was created for the specific needs of certain web developer and the standard classification of suffixes is determined by his needs. You can specify the suffixes in the configuration panel.

Why do we need endings?

The ending of the file is nothing more than an extension of an image file, for example. “jpg”, “.png” or “.svg”.

The range and uniqueness of numbers

By default, the program includes a range of numbers from 100000 to 999999, which corresponds to the decimal number of six digits from the sample of 899999 numbers that provides a good degree of uniqueness of the generated numbers. You can specify a different range in the configuration panel in the upper left corner of the site.

How to use the generator?

To use the generator is very simple. To generate the number simply click the "Get number" button and decimal number with six digits in the range 100000 to 999999 will be generated in the output field. To add a suffix and / or ending to the number, select the desired item in the configuration panel in the upper left corner of the site. You can also add there a suffix, ending or set a custom range of generate numbers. To copy the number click "Copy to clipboard" button next to the number output or push the shortcut keys "Ctrl + C" also you can select the number and copy it by clicking “Copy” item in the context menu.
